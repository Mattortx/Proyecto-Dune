# Indice de Documentacion - Proyecto Dune

## Estructura general

- 01_Gestion_Proyecto
  - 01_Planificacion: cronogramas, scrum, distribucion de practica
  - 02_Equipo: definicion de equipo del proyecto
  - 03_Justificacion: criterios de diseno y modelo jugable
- 02_Documentacion_Tecnica
  - 01_Practicas: documentos base de practica
- 03_Dossieres_Casa_Portil
  - 01_Auditorias: auditorias de continuidad y fe de erratas
  - 02_Maestro: versiones maestras y auditadas
  - 03_Corregidos: versiones corregidas finales/intermedias
  - 04_Fondo_Narrativo: dossier principal y ampliaciones de trasfondo
- 04_Arte_Visual
  - 01_Localizaciones: escenarios y regiones
  - 02_Facciones: unidades, guardias y grupos
  - 03_Heraldica: escudos y paletas oficiales
- 05_Referencia_Dune
  - material de referencia por volumen

## Convencion de nombres

- Formato general: Tipo_Descripcion_YYYYMMDD_HHMMSS.ext
- Sin espacios ni caracteres especiales
- Uso de guion bajo para separacion de palabras
- Numeracion de carpetas para mantener orden estable

## Nota

Esta estructura esta pensada para crecer con nuevas iteraciones sin perder trazabilidad documental.
