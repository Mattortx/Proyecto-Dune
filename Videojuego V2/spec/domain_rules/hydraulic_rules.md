# Domain Rules - Hydraulic Dynasty Manager

## 1. Resource Management
- Resources consumed every "tick" based on population
- Construction requires resources upfront
- Insufficient resources block construction
- Resource generation depends on buildings and staff

## 2. Building System
- 6 categories: Aclima, Exhibition, Science, Logistics, Security, Archive
- Buildings require resources to construct
- Some buildings unlock other buildings (prerequisites)
- Buildings affect resource generation and population

## 3. Population/Creatures
- Population grows with sufficient food/water
- Different roles: Workers, Scientists, Guards, Nobles
- High stress reduces productivity
- Biological state affects population health

## 4. Family System
- Dynasty has ruler and heirs
- Influence affects diplomatic power
- Legitimacy affects stability
- Heirs can become rulers

## 5. Government System
- Ministers manage departments
- Stability affects all operations
- Policies have costs and effects
- Decrees can modify game rules

## 6. Army System
- Internal defense protects against events
- Guards needed for security buildings
- Security level affects risk

## 7. Diplomacy System
- Houses have power and hostility levels
- Reputation affects treaties
- Treaties provide bonuses/penalties

## 8. Event System
- Random events trigger based on conditions
- Events have choices with consequences
- Timeline progresses with decisions

## 9. Palace (Central Hub)
- Main building for all systems
- Unlocks features as level increases
- Hub for family, government, diplomacy

## 10. Game Loop
- No direct player movement
- Decisions through UI tabs
- Construction in building tab
- Events in simulation tab

## 11. Visual Style
- Static background layers (no player movement)
- Atmospheric fog and mist
- Warm interior lighting
- Cool exterior tones
- Reflections on water
